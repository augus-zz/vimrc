## shortcuts

### jump
Ctrl+] => goto definition
Ctrl+t => back to previous position
Ctrl+o => goto last postion

### search
Ctrl+P => find file by filename
Leader + vv => search the the point

### comment

### fold
z+a => toggle fold
z+o => unfold
z+c => fold

### format
= => format the code(block selection)
